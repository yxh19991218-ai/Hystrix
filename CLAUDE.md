## Agent skills

### Issue tracker

Issues live as local markdown files under `.scratch/<feature-slug>/`. See `docs/agents/issue-tracker.md`.

### Triage labels

Five triage roles defined in `docs/agents/triage-labels.md`. All labels use the default names (needs-triage, needs-info, ready-for-agent, ready-for-human, wontfix).

### Domain docs

Single-context repo — one `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
